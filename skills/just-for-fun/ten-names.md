## What it does
Generates 10 name options that aren't the obvious ones, with range across different tones and a quick note on why each works.

## When to use
- Naming a new project, product, feature, or team
- When the obvious name is already taken or too generic
- When you're stuck and need options to react to

## The skill

I need 10 name options for: [what you're naming]

Context:
- What it is / does: [description]
- Who it's for: [audience]
- Tone: [e.g. serious, playful, technical, warm, provocative]
- Names already ruled out: [list]
- Names I like as a reference point: [list, optional]

For each name:
- The name
- One sentence on why it works
- Flag any obvious problems, hard to pronounce, existing trademark, unfortunate connotations

Aim for range across the 10, different tones, lengths, and approaches. No portmanteaus unless they're genuinely good.
